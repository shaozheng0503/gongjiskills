## 附录：在 gongji CLI 里用这些镜像

把上表的镜像地址直接传给 `gongji deploy`：

```bash
# 部署 Qwen3.5-9B
gongji deploy harbor.suanleme.cn/public-hub/qwen35-9b:v0.17.0-rc1 \
  -n qwen-9b -g 4090 -p 8000 --json

# 部署 ComfyUI + Flux.1-dev
gongji deploy harbor.suanleme.cn/public-hub/comfyui-flux:0.5 \
  -n flux-svc -g 4090 -p 8188,3000 --json
```

或者把常用的存成自定义模板：

```bash
gongji images add qwen35-9b \
  --image harbor.suanleme.cn/public-hub/qwen35-9b:v0.17.0-rc1 \
  --gpu 4090 --port 8000 \
  --desc "Qwen3.5-9B vLLM 推理"

gongji deploy --template qwen35-9b -n my-llm --ttl 3600 --json
```
